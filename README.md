# Estética Jazmín
Sistema de citas para salón de belleza desarrollado en Laravel.

## Requisitos
- PHP 8.2+
- Composer
- SQL Server o MySQL

## Instalación
git clone https://github.com/AresGodKiller/Estetica_Web 
cd estetica-jazmin
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
